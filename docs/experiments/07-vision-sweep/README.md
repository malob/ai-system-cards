# Experiment 07 — full-document vision sweep (V1 advisor, first run)

**Question.** What does a vision layer comparing every PDF page render against
its converted markdown find that the mechanical gates cannot? (D22: full 319
pages authorized.)

**Method.** 16 parallel agents, ~20 pages each: Read `extracted/pages/p-NNN.png`
+ the per-page markdown slice (`pipeline/.cache/vsweep/md/`, generated by the
page-slicer from sections-v2), flag content/structure deviations, ignore
styling and known-by-design items. Findings as JSONL per agent;
consolidated here as [findings.jsonl](findings.jsonl). ~1.4M agent tokens.

**Caveat (timing).** The sweep ran against a snapshot taken BEFORE several of
the same night's later fixes (visual-row reconstruction, suspended-hyphen,
ordered-list markers, mark-boundary ordering). Some findings are already fixed;
the re-sweep after the next regen is the clean baseline.

## Result: 134 findings, 69 majors, 40 classes (page lists in worklist.md)

What the sweep caught that NO mechanical gate could see, validating the layer:

- **table-scrambled ×11** (pp. 51–91 cluster, §4 safeguards tables): cell
  reading order/structure wrong in rendered tables — TB1 currently checks
  presence, not cell order. Biggest single class; likely one root cause
  (docling grid traversal vs. my y-slot insertion) — first diagnosis target.
- **transcript-structure ×3 + role-misattributed + content-detached**
  (pp. 107–161): label-only bubbles produce empty turns with content displaced
  into `:::example` blocks; one role flip. Assembler box-mapping class.
- **blockquote-lost ×3 + quote-structure ×2** (pp. 51, 65, 67, 130, 216): UK
  AISI quote blocks flattened — blockquote detection is simply unimplemented;
  the sweep found every instance.
- **caption split/misplaced/duplicated ×7**: caption-continuation merging and
  figure association edge cases.
- **footnote broken/split/merged ×5**: incl. the known stacked-footnote oracle
  boundary (15/16 on p.113) — now with exact instances.
- **suspended-hyphen corruption** ("single**and** multi-agent", p.278) and
  assorted word-merge/spacing — A1 join rule was too eager; **fixed
  same-night** (join now keeps "single- and/or/to" compounds).
- Long tail of 1-instance classes — each a precise, page-located repro.

**False-positive behavior:** good — one agent explicitly verified and
*retracted* its own stranded-bullet findings after checking the full sections
files (page-slicer artifacts, not real). Boundary-spillover guidance in the
prompt worked.

## Conclusions

1. The V1 layer earns its place: 69 precisely-located majors, dominated by
   exactly the classes the gates declaredly don't cover (cell order, box
   semantics, blockquotes, caption association).
2. Sweep findings are *verifier requirements*: each class either becomes a new
   mechanical check (cell-order TB2, blockquote ST4...) or stays V1-owned with
   a written rationale.
3. Re-run the sweep after fixes; track majors-per-sweep as the convergence
   metric toward the H2 review (owner reads only what survives).
